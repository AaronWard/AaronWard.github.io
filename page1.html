About

I have built a Facebook chat bot on the Messenger platform for my 3rd year project. This bot was designed, in a team using the scrum methodology, to help students on campus by assimilating a range of useful articles of information to one place. The bot consisted of the following features:

Features

1- Give real time bus information and college shuttle bus times.
2- Display information about gym and library facilities.
3- Allow students to log into student email/moodle.
4- Reply to the user in a human-like manner.

Additional Information

The bot was developed as a Node.JS app hosted on the Heroku PaaS and was integrated to the Facebook Messenger platform. Three API's were developed for the bus, gym and library services in order to decentralize the application and reduce resource usage. MongoDB was also used to store information about the college facilities. Natural language processing was also utilised to extract context and semantics from the users input. For more information please visit my GitHub.

About 

My final project consisted of a range of technologies including deep learning, facial detection and emotion detection. The core aspects of the project were to train a convolutional neural network to detect facial expression for customer sentiment analysis with TensorFlow.

Features

1 - The AI is trained with a large number of images of human faces.
2 - The AI then learns how to classify the facial expressions and assign them to a certain emotion
3 - The trained TensorFlow model was deployed as an API service, classifies images that it has received and sends back the result.
4- The web application uses Haar cascades for facial tracking and sends cropped images of users face to the model API.

Additional Information

The technologies involved in this project are the TensorFlow machine learning library, Node.js, Heroku PaaS and the Python programming language.
